Running MariaDB Server from extracted zip using cmdline with all defaults
We are using MariaDB 10.4.6 on Windows at port 3307

1. extract the zip file in a location where you have write permission
2. create a data directory inside the <MariaDB-server-path>
3. change directory to <MariaDB-server-path>/bin
4. execute the command "mysql_install_db", this will create all default server tables/dbs with user as root.
5.start the MariaDB server as follows:

mysqld --port=3307 --console

7.once MariaDB server starts successfully connect to the MariaDB server using
mysql -u root --port=3307

8.execute the following command to update root password
ALTER USER 'root'@'localhost' IDENTIFIED BY 'root';

9. shutdown MariaDB server and restart as follows
mysqladmin -u root -p --port=3307 shutdown
mysqld --console --port=3307
